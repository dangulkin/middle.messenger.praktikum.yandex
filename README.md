Messenger page layouts for Yandex Practicum Middle Front-end Developer course 2023-2024

Start using command
$ npm run start

Build using command
$ npm run build

Launch locally on 3000 port using comand
$ npm run start

Check types
$ npm run check-types

Stylelint
$ npm run stylelint

Linting
$ npm run lint
